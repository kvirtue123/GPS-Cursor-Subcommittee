# Skills

Reusable task recipes the agent follows when you invoke them by name — a complete, tested workflow instead of re-explaining the job each time. See the [catalog](../BEST-PRACTICES.md) and [root README](../README.md) for context.

## Where to add

Add each skill as its own folder directly under `skills/`, containing a `SKILL.md` (plus any supporting scripts or assets). **Add your contributions here.**

## Contributing a skill

Open a PR adding your skill folder, or post it in the channel Feed tagged `[Skill]`. The bar: it came from real project work, it's reusable in more than one context, and it has a clear "use this when / not when."
